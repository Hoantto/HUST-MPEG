## 指导
* 为了方便环境的部署, 推荐使用 **conda** 快速构建环境，并在**Linux**内核下完成实验.
* ***步骤***
    > (1) 在本地（推荐）完成实验环境的部署.

    > (2) 修改此行代码```img = Image.open("../ffmpeg-2022-12-11/bin/"+prefix+"."+suffix).convert('RGB')```中的文件路径到指定路径.

    > (3) 编译并运行 **.\mpeg.py** 文件，必要添加代码观察中间结果.
